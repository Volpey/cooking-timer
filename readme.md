# 🍳 CookFlow

CookFlow is a modern cooking timer application built with **Python** and **Flet**.

The app allows users to search for recipes, view cooking information, and start a countdown timer while cooking.

## Features

- Search recipes
- Cooking timer with countdown
- Recipe images
- Recipe overview page
- Modern dark UI
- Modular project structure

## Learning Project

This is only a practice project.

The goal is to learn Python, GUI development, project structure, classes, functions, and modular code organization.

Parts of this project were created with the assistance of AI tools such as ChatGPT. The code was reviewed, changed, and integrated as part of the learning process.
